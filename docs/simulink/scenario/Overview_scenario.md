# Scenario 场景控制模块 (SID:779)

## 模块概述

Scenario模块负责设置仿真环境参数和初始条件，是连接船舶平台(Subsystem)与其他系统的重要接口模块。

## 基本信息

| 属性 | 值 |
|------|-----|
| SID | 779 |
| 内部模块数 | 33 |
| 内部连线数 | 21 |
| 输入端口数 | 6 |
| 输出端口数 | 9 |

| 2-6 | in2-in6 | - | 其他输入信号 |

## 功能说明

Scenario模块的主要职责：

1. **环境参数管理**
   - 设置水流速度和方向
   - 配置风力参数
   - 管理波浪条件

2. **初始条件设置**
   - 船舶初始位置
   - 初始速度和姿态
   - 初始状态向量

3. **仿真场景控制**
   - 场景切换
   - 参数实时调整
   - 边界条件管理

## Ground模块说明

包含6个Ground模块，用于将未使用的输入端口接地，避免仿真警告。
