### 推力曲线方程

螺旋桨转速与推力的关系采用幂律模型：

$$T = \text{sign}(n) \cdot A \cdot |n|^q \times 1000 \quad [\text{N}]$$

其中：
- $T$ : 推力 (N)
- $n$ : 螺旋桨转速 (rpm)
- $A = 8.18448 \times 10^{-5}$ kN/rpm^q
- $q = 2.17020$ (幂指数)

#### MATLAB 实现代码

```matlab
function T_N = rpm2thrust(n_sig)
% rpm2thrust - 将螺旋桨转速转换为推力
%
% 输入:
%   n_sig: 转速信号 (rpm)，可以是向量或标量
%
% 输出:
%   T_N: 推力 (N)，与输入维度一致

    A = 8.18448e-05;   % kN / rpm^q
    q = 2.17020;

    % 如输入是 rad/s，请把下一行打开；如本来就是 rpm，则保持注释
    % n_sig = n_sig * 60/(2*pi);  % rad/s -> rpm

    % 幂律关系
    TkN = sign(n_sig) .* (abs(n_sig).^q) * A;   % 单位: kN
    T_N = 1000 * TkN;                           % 转换为 N

    % -------- 低速死区处理 (逐元素) --------
    % 建立逻辑索引，找到绝对值小于阈值的元素
    idx = abs(n_sig) < 1e-3;
    T_N(idx) = 0;
end
```

## 推进系统特性说明

### 关于螺旋桨推力与航速的关系

#### 问题：为什么 CFD 报告中没有直接给出"螺旋桨推力–航速"曲线？

**核心结论**：

> 螺旋桨推力**不是**航速的唯一函数，而是由**转速 (n)、进速系数 (J)** 及**船体伴流**共同决定的。工程报告采用更科学的表达方式，而非简化的"推力–航速"一对一关系。

**原理解释**：

1. **推力不唯一性**：同一航速下，不同转速会产生不同推力
   - 例如：15 kn 航速，250 rpm 可能产生 30 kN 推力，而 350 rpm 可能产生 60 kN 推力
   - 因此 **航速 → 推力** 不是单值映射

2. **正确的物理关系**：
   ```
   推力 T = f(转速 n, 进速系数 J)
   其中 J = V_A / (n × D)
   V_A = V × (1 - w)  [w 为伴流分数]
   ```

3. **报告的处理方式**：
   - 图 27：给出敞水特性曲线 K_T(J)、K_Q(J)、η(J)（螺旋桨固有特性）
   - 表 4/表 5：在**固定航速**下，扫描不同转速得到推力
   - 图 29：给出"推力–阻力平衡法"预报最大航速

4. **工程关注点**：
   - 不是"某航速下推力是多少"
   - 而是"在可用功率范围内，船能跑多快"
   - 通过 **可用推力 ≥ 阻力** 的交点确定最大航速

#### 问题：能否通过输入转速反推平衡航速？

**答案：可以**，这是标准的"转速反推航速"求解问题。

**求解原理**：

给定转速 n，求解航速 V 使得：
```
T_total(V, n) = R(V)
```

即：推进器在该航速、该转速下的总推力 = 船体阻力

**计算步骤**：

1. **取候选航速 V**：从阻力表（表3）插值得到 R(V)

2. **计算进速**：
   ```
   伴流分数: w = 0.55 × C_B - 0.2
   进速: V_A = V × (1 - w)
   ```

3. **计算进速系数**：
   ```
   J = V_A / (n × D)
   ```

4. **查敞水特性**：从图 27 的 K_T(J) 曲线得到推力系数

5. **计算推力**：
   ```
   单桨推力: T = ρ × n² × D⁴ × K_T(J)
   双桨总推力: T_total = 2T
   ```

6. **力平衡迭代**：
   - 若 T_total > R：增大 V
   - 若 T_total < R：减小 V
   - 用二分法/牛顿法迭代至 T_total = R

**数据支持**：

报告已提供：
- ✅ 阻力数据 R(V)（表3）
- ✅ 敞水特性 K_T(J)、K_Q(J)（图27）
- ✅ 伴流分数计算公式
- ✅ 推进器几何参数（D = 0.810 m）

**实例验证**：

- **12 kn**：表5 显示 198 rpm 时单桨推力 7.88 kN，双桨 ≈ 15.76 kN，接近表3 的阻力 16.09 kN（平衡）
- **19 kn**：表4 显示 327 rpm 为最佳转速，图29 确认极限航速 19.35 kn

**工程应用**：

如需实现"输入转速 → 输出平衡航速"功能，需要：
1. 获取/拟合图27 的 K_T(J) 数据点（约 15-20 个点即可）
2. 实现上述迭代算法
3. 可同时核算功率是否超过主机额定功率（412 kW/台）
