# 船体规格参数

本文档记录本项目仿真船舶的基本规格参数，包括主尺度、船型参数、推进系统配置和推力特性曲线。

## 船舶基本信息

- **排水量**: 90.3 吨
- **螺旋桨配置**: 双螺旋桨，分别距离中轴线垂直距离 1005 mm (即 l1 = l2 = 1.005 m)

### 推力曲线方程

螺旋桨转速与推力的关系采用幂律模型：

$$T = \text{sign}(n) \cdot A \cdot |n|^q \times 1000 \quad [\text{N}]$$

其中：
- $T$ : 推力 (N)
- $n$ : 螺旋桨转速 (rpm)
- $A = 8.18448 \times 10^{-5}$ kN/rpm^q
- $q = 2.17020$ (幂指数)

#### MATLAB 实现代码

```matlab
function T_N = rpm2thrust(n_sig)
% rpm2thrust - 将螺旋桨转速转换为推力
%
% 输入:
%   n_sig: 转速信号 (rpm)，可以是向量或标量
%
% 输出:
%   T_N: 推力 (N)，与输入维度一致

    A = 8.18448e-05;   % kN / rpm^q
    q = 2.17020;

    % 如输入是 rad/s，请把下一行打开；如本来就是 rpm，则保持注释
    % n_sig = n_sig * 60/(2*pi);  % rad/s -> rpm

    % 幂律关系
    TkN = sign(n_sig) .* (abs(n_sig).^q) * A;   % 单位: kN
    T_N = 1000 * TkN;                           % 转换为 N

    % -------- 低速死区处理 (逐元素) --------
    % 建立逻辑索引，找到绝对值小于阈值的元素
    idx = abs(n_sig) < 1e-3;
    T_N(idx) = 0;
end
```
