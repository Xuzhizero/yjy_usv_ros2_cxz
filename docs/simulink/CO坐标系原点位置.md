# CO坐标系原点位置

## 结论

**坐标原点CO位于：船中（Midship）处的水线面上**

具体位置：
- **纵向（x）**：船体中部，距船首和船尾各1米
- **横向（y）**：船体中线
- **垂向（z）**：水线面（z=0为水面）

### 验证依据

1. **纵向对称性**：浮心和重心分别在原点前后0.2米，符合船中作为参考点的习惯
2. **垂向参考**：rg_z = -0.2表示重心在水线下，这是标准做法（水线面为z=0）
3. **横向对称性**：rg_y = 0确认了中线对称，双体船的浮筒对称布置在y = ±0.395m
4. **物理合理性**：对于55kg的船模，重心在水线下0.2米是合理的

这是船舶水动力建模中最常见的坐标系设置方式。

## 推理过程

### 关键线索分析

#### 线索1：船体尺寸
- L = 2 m（船长）
- B = 1.08 m（船宽）
- 这是一个小型船模

#### 线索2：浮心位置
- LCF = -0.2 m（浮心纵向位置）
- 文档说明是"相对参考点"
- **负值意味着浮心在原点后方0.2米**

#### 线索3：重心位置
- rg = [0.2, 0, -0.2]ᵀ
- rg_x = 0.2 m → **重心在原点前方0.2米**
- rg_y = 0 m → **重心在中线上（对称）**
- rg_z = -0.2 m → **重心在原点下方0.2米**

#### 线索4：浮筒布置
- y_pont = 0.395 m（浮筒横向位置）
- 双体船配置，浮筒对称布置在 y = ±0.395 m

### 位置推断

从浮心和重心的相对位置关系：
- 浮心：x = -0.2 m（船中后方10%船长）
- 重心：x = +0.2 m（船中前方10%船长）
- **两者相距0.4米，约占船长的20%**

这个配置强烈暗示坐标原点位于船中（Midship）处。

### 坐标系设置的物理意义

1. **船中作为原点**是船舶工程中最常见的做法，便于前后对称分析
2. **水线面为z=0**是标准的垂向参考，便于计算水动力
3. **重心在水线下**（z=-0.2）符合船体重量分布的物理规律
4. **浮心略微在船中后方**是常见的设计，有利于稳定性

### 验证合理性

- 对于55kg、2米长的船模
- 重心在水线下0.2米（10%船长）
- 重心略微在船中前方0.2米（10%船长）
- 浮心在船中后方0.2米（10%船长）

这些参数都在合理范围内，符合双体船的典型设计特征。
