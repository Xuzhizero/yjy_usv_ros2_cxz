# simulink2UE1 模块 (SID:751)

## 模块概述

simulink2UE1是备用的虚幻引擎通信接口，结构比主接口simulink2UE5更复杂，包含更多的配置选项和处理逻辑。

| 内部模块数 | 47 |

| 输入端口数 | 3 |
| 嵌套子系统数 | 2 |
