# ROS2simulink 模块 (SID:10)

## 模块概述

ROS2simulink是主要的ROS2输入处理模块，负责从ROS2系统接收数据，进行解析和分配，主要处理姿态角和转速信息。

## 基本信息

| 属性 | 值 |
|------|-----|
| SID | 10 |
| 内部模块数 | 18 |
| 内部连线数 | 9 |
| 输出端口数 | 3 |

## 内部模块构成

| 模块类型 | 数量 | 功能 |
|---------|------|------|
| Display | 4 | 数值显示/调试 |
| Reference | 3 | 库引用模块 |
| Constant | 3 | 常数源 |
| Outport | 3 | 输出端口 |
| BusSelector | 2 | 总线信号选择 |

## 输入/输出

### 输入
- 从ROS2 Topic订阅的消息数据

### 输出

| 端口 | 信号名 | 说明 | 连接目标 |
|------|--------|------|----------|
| 1 | psi | 姿态角/偏航角 | simulink2UE5 in:1 |
| 2 | n | 转速信息 | simulink2UE5 in:2 |
| 3 | signal3 | 第三信号 | simulink2UE5 in:3 |
