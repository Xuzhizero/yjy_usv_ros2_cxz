# ROS2simulink1 模块 (SID:813)

## 模块概述

ROS2simulink1是辅助的ROS2输入处理模块，处理额外的ROS2数据流，包含数据类型转换功能。

## 基本信息

| 属性 | 值 |
|------|-----|
| SID | 813 |
| 内部模块数 | 8 |
| 内部连线数 | 5 |
| 输出端口数 | 2 |

## 内部模块构成

| 模块类型 | 数量 | 功能 |
|---------|------|------|
| DataTypeConversion | 2 | 数据类型转换 |
| Display | 2 | 数值显示/调试 |
| Outport | 2 | 输出端口 |
| BusSelector | 1 | 总线信号选择 |
| Reference | 1 | 库引用模块 |

## 输入/输出

### 输入
- 从ROS2 Topic订阅的消息数据

### 输出

| 端口 | 信号名 | 说明 | 连接目标 |
|------|--------|------|----------|
| 1 | out1 | 辅助数据1 | Bus Creator1 in:1 |
| 2 | out2 | 辅助数据2 | Bus Creator1 in:2 |
