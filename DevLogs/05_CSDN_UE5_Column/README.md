## 使用说明

### 本地浏览

所有文章已转换为 Markdown 格式，可以使用任何支持 Markdown 的编辑器或查看器阅读。

### 文件命名规则

```
YYYY-MM-DD__<slug>.md
```

- `YYYY-MM-DD`: 文章发布日期
- `<slug>`: 基于标题生成的 URL 友好标识符

### Frontmatter 格式

每篇文章顶部包含 YAML frontmatter：

```yaml
---
title: 文章标题
date: 发布日期
source: CSDN 原文链接
tags: [标签1, 标签2, ...]

---
```
